# StreamPOO

Petit projet PHP oriente objet autour d'un catalogue de streaming.

## Fonctionnalites

- Classe abstraite `Video`
- Classes `Film`, `Serie` et `Episode`
- Catalogue avec recherche et filtre par genre
- Playlist utilisateur
- Page responsive en PHP natif

## Lancer le projet

```bash
php -S 127.0.0.1:8000
```

Puis ouvrir `http://127.0.0.1:8000` dans le navigateur.
